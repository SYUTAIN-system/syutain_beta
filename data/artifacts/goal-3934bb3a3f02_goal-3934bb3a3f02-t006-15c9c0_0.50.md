# 商品説明文：Booth向けオリジナル素材集「PixelCraft Pro」

## 商品概要
「PixelCraft Pro」は、実務クリエイター向けに開発された高品質ピクセルアート素材集です。ゲーム開発、Webデザイン、印刷物制作など、多様なプロジェクトで即戦力となる2000点以上の素材を収録しています。

## 利用シーン

### 制作のきっかけ
多くのクリエイターから「商用プロジェクトで使える高品質なピクセルアート素材が少ない」「一貫性のあるスタイルの素材集が欲しい」という声を受けて開発しました。特にインディーゲーム開発者や個人クリエイターが、限られた時間と予算でプロフェッショナルな品質を実現できるよう設計されています。

### 活用方法
1. **ゲーム開発**: RPGのキャラクター、アイテム、タイルセットとして
2. **Webデザイン**: レトロ風サイトのUI要素、装飾アイコンとして
3. **印刷物**: フライヤーやパッケージのアクセントグラフィックとして
4. **動画制作**: モーショングラフィックスの素材として

## ライセンスの注意点

### 許可される使用範囲
- 個人・商用プロジェクトでの使用
- ゲームやアプリケーションへの組み込み
- 編集・改変による派生作品の制作
- Boothでの再販（条件付き）

### 制限事項
- 素材単体での再配布・転売は禁止
- 競合素材集としての再パッケージングは禁止
- 違法・公序良俗に反するコンテンツでの使用は禁止

### Booth再販について
本素材を組み込んだ作品（ゲーム、イラスト集等）の販売は可能ですが、素材そのものや「素材集」としての再販はできません。詳細なライセンス条項は同梱の「license.pdf」をご確認ください。

## 類似作との差別化ポイント

### 1. 実用性重視の設計
- **一貫したスタイル**: 全ての素材が同じ色パレット・スタイルで統一されているため、組み合わせて使用しても違和感がありません
- **モジュラー設計**: キャラクターパーツは分割されており、組み替えてオリジナルキャラクターが作成可能
- **アニメーション対応**: 主要キャラクター・エフェクトには8方向アニメーションを収録

### 2. プロ仕様の機能性
- **レイヤー分けPSDファイル**: 主要素材は編集可能なPSD形式で提供
- **規格の統一**: 全てのタイル素材は32x32px基準で設計、ゲームエンジンでの使用が容易
- **メタデータ充実**: 色数、推奨使用シーン、編集時の注意点を明記

### 3. 継続的アップデート
- 購入者には無料アップデートを保証（年2回の大規模追加予定）
- ユーザー要望に基づく素材の追加リクエスト受付

### 4. サポート体制
- 商用利用に関する個別相談対応
- 主要ゲームエンジン（Unity, Godot等）へのインポートチュートリアル提供
- アクティブなユーザーコミュニティへの招待

## 技術仕様
- 解像度: 32x32px基準（一部素材は64x64px）
- ファイル形式: PNG, PSD, Aseprite（.aseprite）
- 総ファイル数: 2,150点
- 推奨ソフト: Aseprite, Photoshop, Clip Studio Paint

「PixelCraft Pro」は、単なる素材集ではなく、クリエイターの制作プロセスを加速する「創作パートナー」として開発されました。品質と実用性のバランスにこだわり、プロジェクトの完成度を向上させる真のツールを目指しています。
