# Booth向け商品コンセプト：手作り感のある「植物育成キット」

## 商品コンセプト
「自分で育てる小さな緑の世界」をテーマにした、観葉植物の種子と手作りの陶器鉢がセットになった育成キット。完成品ではなく「育てる過程」に価値を見出す層をターゲットにしています。

## 制作の実際

### 必要な材料と調達
- **陶器鉢制作**：陶芸用粘土（1キロ約800円）、釉薬セット（3,000円程度）、窯焼き設備（自宅にない場合は窯レンタル1回2,000〜4,000円）
- **植物要素**：観葉植物の種子（100粒で1,500円程度）、培養土（5Lで500円）、育て方説明カード（印刷代含む）
- **包装資材**：化粧箱（1個150円）、緩衝材、ラッピング素材

### 制作工程と時間
1. **陶器成形**：1個あたり成形時間20分＋乾燥期間2〜3日
2. **窯焼き**：素焼き（800℃で6時間）→釉薬付け→本焼き（1,200℃で8時間）※窯の容量により一度に焼ける数に制限あり
3. **セット組み立て**：土の計量・袋詰め、種子の小分け、説明書作成・印刷
4. **パッケージング**：1セットあたり15分程度

### 初期投資と単価計算
- **設備投資**：窯がない場合、小型電気窯で10万円〜
- **材料費/1セット**：約450円（鉢・土・種子・包装含む）
- **制作時間**：アクティブな作業時間で1セットあたり約45分（乾燥・焼成時間除く）
- **想定販売価格**：1,800円〜2,200円（Booth手数料20%考慮）
